Para empezar a usar la página web, lo primero que debes hacer es crear una cuenta, que se guardará en la base de datos.



Si ya tienes una cuenta creada, solo tendras que meter el correo electrónico y la contraseña.



Una vez el usuario este registrado, le aparecerán las categorías disponibles.



Cuando selecciones una de las categorías, le apareceran los productos disponibles.



Al pulsar el botón comprar, te trasladará a la página donde este disponible el producto.



Y para salir, lo único que debes hacer es dar a cerrar sesión, y te llevara a la página del principio.
